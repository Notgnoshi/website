We're getting the gang back together!

We've been playing together for years (at least since version 1.2 for some of us!), and we've followed each other across many different servers over that time. We've built a community! We've suffered through the terrifying first nights together, conquored dragons together, and laughed together as we refused to help each other fight off hordes of Zombie Pigmen.

Lately though, we've had to jump from server to server, each time starting from scratch. And as time has progressed, we've all gotten busier, to the point that we no longer play together. This is an attempt to fix that.

Pending the official release of Minecraft 1.13, we'll be hosting a vanilla Minecraft server right here at mc.agill.xyz! I'll send out texts, emails, Reddit messages, and smoke signals once we're up and running, but for now get your creative juices flowing!

Why start over again?

Minecraft 1.13 is almost here, and it's looking sweet! I'm hoping to combined excitement of trying to get us back together, and exploring a new update together is enough to get people going again.

I'm also looking to get something semi-permanent set up. I already own a server for other things, so I can run a Minecraft server long-term without having to pay for a VPS.

I also think using vanilla is also a good reason to start over. Read on!

Why vanilla?!

I know, I know, we love our plugins. But using plugins is the single biggest reason maintaining a server is difficult. Besides, we've always advertised ourselves as "semi-vanilla", and have used plugins to supplement, not replace vanilla functionality.

Here's a few of my thoughts:

  • A vanilla server is far easier to maintain.
  • We can update to a new Minecraft version the week it comes out.
  • We can get close to a /home and/or /warp experience with command blocks. We could also contract some certain road-building fiend to build us a Nether network.
  • Grief Prevention? We're all adults. If it becomes a problem, we can handle it at that time. I've thought about whitelisting the server in an attempt to mitigate this risk, but haven't come to a conclusion yet. I'd welcome your input!
  • What about my beloved Chest Shops? Most of us have resorted to bartering anyways.
  • I miss the challenge! Sure, flying was cool, but after a while, I had a chest (or 32) dedicated to every item in the game.

Let me know what you think, and start getting hyped!
